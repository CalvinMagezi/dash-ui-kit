# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup
- Core CSS utility system with Tailwind-like classes
- Design token system with CSS variables
- Button component with multiple variants (default, outline, ghost, destructive)
- Card component with sub-components (Header, Title, Description, Content, Footer)
- Input component with label and error states
- Select/Dropdown component
- Badge component with variants
- Comprehensive documentation
- Unit and integration tests
- CI/CD pipeline with GitHub Actions

## [0.1.0] - 2025-11-07

### Added
- First alpha release
- MVP feature set complete
- Published to PyPI
