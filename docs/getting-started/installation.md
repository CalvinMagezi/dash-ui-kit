# Installation

## Requirements

- Python 3.8 or higher
- pip

## Installing from PyPI

The easiest way to install Dash UI Kit is via pip:

```bash
pip install dash-ui-kit
```

## Installing from Source

To install from source (for development):

```bash
# Clone the repository
git clone https://github.com/CalvinMagezi/dash-ui-kit.git
cd dash-ui-kit

# Install in development mode
pip install -e ".[dev]"
```

## Verifying Installation

To verify that Dash UI Kit is installed correctly:

```python
import dash_ui_kit
print(dash_ui_kit.__version__)
```
